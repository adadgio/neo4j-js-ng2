Edit: Node & link edition

Simple queries let you find nodes without typing long cypher queries.

// pseudo code format
:Label1:Label2 property="Value" limit,skip // using numbers, limit and multiple properties
:Person name="Ben" age=12 10 // using everything
:Person name="Ben" age=34 50,0

Display one level of relationships using the "+1" flag at the very end of the query

:Company name="Gougle" +1

Create mode lets you create new nodes and relationships. You can toggle the create mode by clicking on the create mode switch ().

You can also toggle create mode by using the Ctrl+C shortcut.

Long-click on the empty circle to create a new node, and drag a line between two nodes to create a relationship.
